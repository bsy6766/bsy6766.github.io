Volt2D



Detail

This project is a very simple 2D Game Framework & Engine started from my graphic course's final project in college. Supports many features for 2D game development such as sprite, sprite animation, progress timer (both bar and clock-ish type), text rendering, action schedule, particle system, sound system, shaders and more.

In the beginning, the main purpose of this project was to learn how to use OpenGL. Then I extended a lot more by adding various features and learned much more about game engine and other techniques, like text rendering, shader, reducing draw calls, etc. This project is no longer active.

Language, Engine & Tools

C++, OpenGL, FMOD, FreeType2, PhotoShop, GLFW, GLM, Lua

Date

2015

Meteor Tap



Detail

This is a simple 2D mobile game that you touch to meteor to destroy before it hits your planet. Each planet has their own special ability to defend itself and different types of meteor starts to attack as game goes harder. There are 3 different modes to play and supports Apple's Game Center.

This project was a 3 months long project during my summer break while I was in college and everything in this game (code, art, sound effect, music, level design) were made by me. The game is not on the Apple's Appstore anymore but still exists and runs in my iphone.

Language, Engine & Tools

Objective-C, cocos2d-iphone, PhotoShop, TexturePacker

Head Manager



Detail

This is a game Minecraft's server plugin that runs with Spigot. The main feature of this plugin was to let players access to 2000+ head blocks in minecraft. Not only just blocks, it also had in-game virtual shop where players can browse, purchase and sell head blocks. Server admin can customize the in-game shop and every little detail settings in plugin.

The plugin was a paid resource at Spigot website. It supported upto Minecraft version 1.8.3.
